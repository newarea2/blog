# VanBlog

## 1 执行脚本

```sh
curl -L https://vanblog.mereith.com/vanblog.sh -o vanblog.s
```

![](https://image.newarea.site/2024-04-16-23-16-23.png)

## 2 放行端口

进入宝塔面板，使防火墙和轻量云防火墙放行上述配置的端口 3000、3002。


## 3 初始化

在浏览器中访问 `http://122.51.210.64:3000/admin/init`
